# memo — Локальная AI-система для работы с документами

## Описание проекта

**memo** — десктопное приложение для локальной работы с документами со встроенным AI-ассистентом. Все вычисления выполняются на машине пользователя через Ollama — данные не покидают устройство.

Проект разработан в рамках учебного курса «Проектный практикум» (МТУСИ, группа БВТ2403).

---

## Проблематика

Современные AI-инструменты для работы с документами (NotebookLM, ChatGPT и аналоги) требуют передачи данных на внешние серверы. Это создаёт риски для конфиденциальности при работе с личными, корпоративными или чувствительными документами.

**memo** решает эту проблему: все операции — индексация, поиск, генерация — выполняются локально, без подключения к интернету.

---

## Целевая аудитория

Универсальный пользователь с личной или рабочей базой документов, которому необходимо:
- искать информацию по документам
- упорядочивать файлы без ручной сортировки
- создавать новые документы с помощью AI

---

## Ключевые возможности

### 1. Q&A по документам (RAG)
Пользователь задаёт вопрос — AI отвечает на основе содержимого выбранных документов с указанием источников (названий файлов).

### 2. Авто-организация файлов
AI анализирует документы и предлагает структуру папок по семантике содержимого. Пользователь редактирует предложенную структуру и подтверждает — файлы перемещаются.

### 3. Создание документов
AI генерирует новые документы (.txt или .md) по текстовому описанию пользователя. Поддерживает plain text и Markdown-форматирование.

### 4. Индексация документов
Пользователь управляет индексом вручную: выбирает файлы и папки, запускает индексацию. Система отслеживает изменения файлов и уведомляет об устаревшем индексе.

## Интерфейс

Двухпанельный layout:

| Левая панель | Правая панель |
|---|---|
| Файловый менеджер | AI-ассистент |
| Дерево папок и файлов | Чат (Q&A) |
| Навигация по директориям | Создание документов |
| Статусы индексации | Список чатов |
| Превью авто-организации | — |

---

## Поддерживаемые форматы документов

- PDF (.pdf)
- Word (.docx)
- Markdown (.md)
- Plain text (.txt)

---

## AI-модели

Система использует Ollama для запуска моделей локально.

| Функция | Модель |
|---|---|
| Q&A, создание документов | Выбирается пользователем из установленных в Ollama |
| Авто-организация файлов | Предустановлена, не требует настройки |

---

## Ключевое преимущество

**Полная локальность и конфиденциальность** — данные пользователя не покидают устройство. Приложение работает без подключения к интернету.
