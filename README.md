# memo

Локальная AI-система для работы с документами. Десктопное приложение с встроенным AI-ассистентом — все вычисления выполняются на машине пользователя через Ollama, данные не покидают устройство.

> Учебный проект. Курс «Проектный практикум», МТУСИ, группа БВТ2403.

---

## Возможности

- **Q&A по документам** — AI отвечает на вопросы по содержимому выбранных файлов и указывает источники
- **Авто-организация файлов** — AI предлагает структуру папок по семантике документов
- **Создание документов** — генерация новых `.txt` и `.md` файлов по запросу
- **Индексация** — ручное управление индексом с отслеживанием изменений файлов

## Поддерживаемые форматы

PDF · DOCX · Markdown · TXT

## Стек

| Слой | Технология |
|---|---|
| Десктоп-оболочка | Tauri 2 |
| UI | React + TypeScript |
| Backend | Python 3.11+ / FastAPI |
| RAG | ChromaDB (собственный конвейер чанкинга и retrieval) |
| LLM | Ollama (локально) |

## Требования

- [Ollama](https://ollama.com) — должна быть установлена и запущена
- Node.js 18+
- Python 3.11+
- Rust (для Tauri)

### Модели Ollama

Перед запуском установите необходимые модели:

```
ollama pull bge-m3          # эмбеддинги (обязательно)
ollama pull qwen3:1.7b      # авто-организация (обязательно)
ollama pull qwen3:4b        # рекомендуемая модель чата
```

## Разработка

### Первый запуск

```powershell
# Backend — создать venv и установить зависимости
cd src\backend
python -m venv .venv
.venv\Scripts\pip install -e ".[dev]"

# Frontend — установить npm-пакеты
cd ..\frontend
npm install
```

### Запуск в dev-режиме

```powershell
# из src/frontend/
npm run tauri dev
```

Tauri автоматически запускает Vite (порт 1420) и Python-backend. Порт backend выбирается динамически.

### Тесты

```powershell
cd src\backend
.venv\Scripts\python -m pytest tests\unit -v
```

### Переменные окружения backend

| Переменная | По умолчанию | Описание |
|---|---|---|
| `MEMO_OLLAMA_URL` | `http://localhost:11434` | URL Ollama |
| `MEMO_DATA_DIR` | `./data` | Папка БД и ChromaDB |
| `MEMO_EMBED_MODEL` | `bge-m3` | Модель эмбеддингов |
| `MEMO_NAME_MODEL` | `qwen3:1.7b` | Модель для авто-названий |
