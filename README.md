# memo

Локальная AI-система для работы с документами. Десктопное приложение с встроенным AI-ассистентом — все вычисления выполняются на машине пользователя через Ollama, данные не покидают устройство.

> Учебный проект. Курс «Проектный практикум», МТУСИ, группа БВТ2403.

---

## Возможности

- **Q&A по документам** — AI отвечает на вопросы по содержимому выбранных файлов и указывает источники
- **Авто-организация файлов** — AI предлагает структуру папок по семантике документов
- **Создание документов** — генерация новых `.txt` и `.md` файлов по запросу
- **Индексация** — ручное управление индексом с отслеживанием изменений файлов

## Поддерживаемые форматы

PDF · DOCX · Markdown · TXT

## Стек

| Слой | Технология |
|---|---|
| Десктоп-оболочка | Tauri 2 |
| UI | React + TypeScript |
| Backend | Python 3.11+ / FastAPI |
| RAG | LangChain + ChromaDB |
| LLM | Ollama (локально) |

## Требования

- [Ollama](https://ollama.com) — должна быть установлена и запущена
- Node.js 18+
- Python 3.11+
- Rust (для Tauri)

### Модели Ollama

При первом запуске приложение предложит установить:

```
ollama pull bge-m3          # эмбеддинги (обязательно)
ollama pull qwen3:1.7b      # авто-организация (обязательно)
ollama pull qwen3:4b        # рекомендуемая модель чата
```
